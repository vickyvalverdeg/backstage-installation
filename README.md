# backstage-installation
Backstage Installation

## Requirements
- yarn >=1.22.17
- node >=14.17.0

## CLI
### Install standalone application locally
`npx @backstage/create-app`

### Installing lib dependencies
`yarn install`

### Run locally
Run from inside app directory: `yarn dev`
